# Curso de Programação Java - Senai Roberto Simonsen

Bem-vindo ao repositório do curso de Programação Java oferecido pelo Senai Roberto Simonsen. Aqui você encontrará todos os materiais e projetos desenvolvidos durante o curso de 120 horas, que foi projetado para ajudar a aprender Java de forma prática e completa.

# Objetivo do Curso

O curso tem como objetivo te ensinar a desenvolver sistemas usando Java, cobrindo desde os fundamentos de programação até conceitos avançados de Orientação a Objetos e desenvolvimento de interfaces gráficas.

# Tecnologias Utilizadas
* Linguagem principal: Java
* IDE: IntelliJ IDEA, Eclipse
* Frameworks: JavaFX (para interfaces gráficas), Spring (para aplicações web)
* Banco de Dados: MySQL, SQLite
* Controle de Versão: Git e GitHub

# Projetos desenvolvidos 
* Calculadora ✅
* Sistema Bancario ✅
